# RFM Dashboards (Estático + Interactivo)

Este paquete incluye:
- dashboard_estatico.py -> Dashboard con gráficos Seaborn/Matplotlib y estrategias.
- dashboard_interactivo.py -> Dashboard con gráficos interactivos (Plotly).

## Instalación:
pip install streamlit pandas matplotlib seaborn openpyxl plotly scipy

## Ejecución:
streamlit run dashboard_estatico.py
streamlit run dashboard_interactivo.py
